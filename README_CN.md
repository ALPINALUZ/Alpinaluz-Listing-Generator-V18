# Alpinaluz Listing Generator V18.2.8

V18.2.8 基于 V18.2.7，不新增“群里上报卡”，继续保持新手截图上报流程。

本版重点：

- 固定五点顺序模板，让正文结构更像成熟 Amazon listing。
- 中文解释改为“精简但完整”：不强制字数，但不能漏产品类型、材质/外观、灯头、尺寸、功能、风险判断等关键点。
- 天然木 / 手工编织不再过度保守；只有明确 MDF、仿木、热转印木纹、机器编织等情况才降级表达。
- 保留新手一键确认、推荐卡、字数提示、风险提示、Token CSV 导出。
- 保留空正文拦截，不完整国家不导出。

新手标准流程：

1. 填产品资料并生成事实卡。
2. 生成 ES 推荐标题和商品亮点。
3. 截图给主管看；没有明显问题就一键确认。
4. 生成多国语言标题并一键确认。
5. 生成正文完整包。
6. 导出 TXT 和 Token CSV 给主管最终审核。

运行：
Mac：双击 `run.command`
Windows：双击 `run_windows.bat`
