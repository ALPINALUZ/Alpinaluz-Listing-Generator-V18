# Alpinaluz Listing Generator V18.2.7

V18.2.7 以 V18.2.6 文案逻辑为基准，主要优化 UI：

- 新手默认只看 AI 推荐卡，不必理解全部字段。
- 每个国家提供“一键确认当前推荐标题+亮点”。
- 高级编辑、重生标题、回退等功能默认折叠，给主管或熟手使用。
- 预览区增加“主管审批摘要”，方便主管快速看标题长度、亮点长度、五点/A+完整性和核心亮点。
- 保留完整中文解释、Natural Wood 安全规则、空正文拦截、Token CSV 导出。

运行：
Mac：双击 `run.command`
Windows：双击 `run_windows.bat`
